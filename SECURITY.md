# Security Policy

## Supported Versions

Currently, Cavrix AI is in active rolling release. Only the latest ISO release and up-to-date installed systems are supported with security updates.

| Version | Supported          |
| ------- | ------------------ |
| Latest  | :white_check_mark: |
| Older   | :x:                |

## Reporting a Vulnerability

Security is a top priority for Cavrix Core Technologies. 

If you discover a security vulnerability in the custom Cavrix AI packages (`cavrix-welcome`, `cavrix-ai`, `cavrixos-branding`), the installer scripts, or the ISO generation pipeline, please **DO NOT** open a public issue.

Instead, please email the security team at **security@cavrixos.org**. 

We will acknowledge your report within 48 hours and work with you to remediate the vulnerability before public disclosure.

*(Note: If the vulnerability is within an upstream package like the Linux Kernel or KDE Plasma, please report it directly to the upstream maintainers).*
