# Installing CavrixOS

1. Download the latest `.iso` release.
2. Flash the ISO to a USB drive using a tool like Rufus, BalenaEtcher, or `dd`.
3. Boot from the USB drive.
4. If you boot into the graphical environment (KDE Plasma), an "Install CavrixOS" shortcut will be available on the desktop/autostarted.
5. If you boot into the console, run `cavrixos-installer` to start the installation.
6. Follow the guided prompts to select your disk, set up encryption (optional), and create your user account.
7. Reboot into your new CavrixOS installation!
