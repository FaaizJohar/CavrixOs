# Support Guidelines

## Getting Help

If you require assistance with CavrixOS, please utilize the following resources:

1. **Official Documentation**: Check the `docs/` folder in the repository or our official website.
2. **Community Forums**: Join the CavrixOS Discourse server at `forum.cavrixos.org`.
3. **Arch Wiki**: Since CavrixOS is an Arch derivative, the [ArchWiki](https://wiki.archlinux.org/) is directly applicable to 99% of system-level issues.
4. **Issue Tracker**: Use the GitHub Issues page *only* for verified bugs or feature requests, not for general technical support.

## When Opening an Issue

If you must open an issue, ensure you provide:
- The exact version of the CavrixOS ISO used during installation.
- The output of `journalctl -xe` if reporting a crash.
- Detailed reproduction steps.
